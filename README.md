# EveryPage Quran | QuranSRT

**EveryPage Quran (QuranSRT)** is an advanced AI-powered tool designed to generate synchronized subtitles (SRT) for Quran recitations automatically. It fetches data from the Quran Cloud API and creates perfectly timed subtitles in Arabic and various translations, ready for video editing.

![App Screenshot](https://via.placeholder.com/800x400?text=EveryPage+Quran+App)

## ‚ú® Features
- **Auto-Sync**: Automatically calculates audio duration for precise subtitle timing.
- **Multi-Reciter**: Support for 40+ world-famous Qaris (reciters).
- **Multi-Language**: Translations available in English, Indonesian, French, and 20+ other languages.
- **Smart Splitting**: Advanced text splitting logic (Verse by Verse / Waqaf / Standard).
- **Dark/Light Mode**: Beautiful, high-contrast UI adaptable to your preference.
- **Privacy Focused**: No data is stored on our servers; everything is processed on the fly.
